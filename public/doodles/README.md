## Image Guidelines

- **Format**: PNG (with transparent background recommended)
- **Size**: Around 100x100 to 200x200 pixels works well
- **Style**: Comical hand-drawn doodles

## How It Works

- Each doodle has a subtle wiggle animation for a less static feel
- Clicking on a doodle takes the user to that person's Wikipedia page or website
